1. หัวข้อและคำอธิบาย (Project Title & Description)
#  ระบบจองโต๊ะอาหารออนไลน์ (Restaurant Table Booking System)

เว็บแอปพลิเคชันสำหรับจองโต๊ะอาหารและจัดการเมนู พัฒนาด้วย Django Framework ออกแบบมาเพื่อให้ลูกค้าสามารถจองโต๊ะล่วงหน้าได้อย่างสะดวก และช่วยให้ผู้ดูแลระบบ (Admin) จัดการร้านอาหารได้อย่างมีประสิทธิภาพ

2. ฟีเจอร์หลัก (Key Features)

### ฝั่งผู้ใช้งาน (User / Customer)
* **ระบบสมาชิก:** สมัครสมาชิกและเข้าสู่ระบบเพื่อทำการจอง
* **การจองโต๊ะ:** เลือกโต๊ะที่ว่างตามวันที่และเวลาที่ต้องการ (ระบบจะกรองเฉพาะโต๊ะที่รองรับจำนวนคนและสถานะพร้อมใช้งาน)
* **จัดการการจอง:** ดูประวัติการจอง แก้ไขข้อมูล หรือยกเลิกการจองของตนเองได้
* **ดูเมนูอาหาร:** แสดงรายการเมนูอาหารที่พร้อมขาย พร้อมระบบค้นหาและกรองตามราคา

### ฝั่งผู้ดูแลระบบ (Admin)
* **Dashboard:** หน้าแดชบอร์ดสรุปยอดการจอง, จำนวนโต๊ะที่ว่าง, และเมนูอาหารทั้งหมด
* **จัดการการจอง:** ตรวจสอบและอัปเดตสถานะการจอง (เช่น รอการยืนยัน, ยืนยันแล้ว, ยกเลิก)
* **จัดการโต๊ะ:** เพิ่ม/ลบ โต๊ะ และอัปเดตสถานะโต๊ะ (เปิด/ปิดใช้งานชั่วคราว) ซึ่งจะส่งผลต่อหน้าเว็บของลูกค้าทันที
* **จัดการเมนูอาหาร:** เพิ่ม ลบ และอัปโหลดรูปภาพเมนูอาหาร พร้อมปุ่มสลับสถานะ (พร้อมขาย/หมดชั่วคราว)

### ระบบเบื้องหลัง (System Logic)
* **Smart Redirect:** นำทางผู้ใช้ไปยังหน้าที่เหมาะสมทันทีหลังเข้าสู่ระบบ (Admin เข้าหน้า Dashboard / User เข้าหน้าหลัก)
* **Validation & Security:** ป้องกันการจองโต๊ะซ้ำในวัน/เวลาเดียวกัน และไม่สามารถจองย้อนหลังได้


3. โครงสร้างโปรเจกต์ (Project Structure)
myproject/
├── image/               # โฟลเดอร์เก็บรูปภาพสำหรับแสดงในไฟล์ README
├── media/               # โฟลเดอร์เก็บรูปภาพเมนูอาหารที่ผู้ดูแลระบบอัปโหลด
├── myapp/               # โฟลเดอร์แอปพลิเคชันหลักของระบบ
│   ├── static/          # โฟลเดอร์เก็บไฟล์ CSS, JavaScript และรูปภาพตกแต่งเว็บ
│   ├── templates/       # โฟลเดอร์เก็บไฟล์หน้าเว็บ HTML ทั้งหมด
│   ├── models.py        # โครงสร้างตารางฐานข้อมูล (Table, Booking, FoodMenu)
│   └── views.py         # โฟลเดอร์เก็บระบบตรรกะและฟังก์ชันการทำงานหลัก
├── myproject/           # โฟลเดอร์เก็บการตั้งค่าหลักของโปรเจกต์ (มีไฟล์ settings.py)
├── db.sqlite3           # ไฟล์ฐานข้อมูล SQLite
├── manage.py            # ไฟล์รันคำสั่งหลักของระบบ Django
└── README.md            # คู่มืออธิบายการใช้งานระบบ

4. การติดตั้งและการตั้งค่า (Installation & Setup)


Clone โปรเจกต์: 
git clone https://github.com/thanawetphisitcha-dotcom/project.git
cd myproject

สร้าง Virtual Environment: 
python -m venv venv
venv\Scripts\activate      # สำหรับ Windows
# source venv/bin/activate # สำหรับ Mac/Linux

ติดตั้ง Library ที่จำเป็น: 
pip install django

ตั้งค่าฐานข้อมูล: 
python manage.py makemigrations
python manage.py migrate

สร้างบัญชีแอดมิน: 
python manage.py createsuperuser

รันเซิร์ฟเวอร์: 
python manage.py runserver

### การตั้งค่าเพิ่มเติม (Configuration)

การเปลี่ยน TIME_ZONE เป็น Asia/Bangkok เพื่อให้เวลาการจองตรงกับไทย

การตั้งค่า MEDIA_ROOT เพื่อรองรับการอัปโหลดรูปภาพเมนูอาหาร
